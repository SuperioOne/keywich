# Shortcuts

**General:**

- `Ctrl` + `1` : Open home page.
- `Ctrl` + `2` : Open keys list.
- `Ctrl` + `3` : Open settings.
- `Ctrl` + `L` : Lock/Logout.
- `Ctrl` + `I` : Log panel.

**Keys page:**

- `Ctrl` + `S` : Open/Close key filter.
- `Ctrl` + `N` : Create new key.

> Current version does not have configurable shortcuts.
